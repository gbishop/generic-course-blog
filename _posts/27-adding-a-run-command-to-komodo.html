You can make your life ton's easier if you add a command to Komodo to automatically save your file and run the script you are editing. I'll show you how.

First open Komodo Edit and click the View menu and select Tabs & Sidebars and select Toolbox. Komodo should look something like this:

Now click on the little arrow next to the gear in the Toolbox. It is the Tools button and select New Macro. You should get a dialog that looks like this:

Fill in the first box so it says "Run Python Script". The copy and paste the lines below for your OS into the second box. The dialog should look something like this:

Now click on the Key Binding tab and click in the box labeled New Key Sequence. Press the F5 key on your keyboard; F5 should appear in the box. Now press the Add button. The dialog should look something like this:

Now click OK. You should be able now to open a python script (say A4.py) and hit F5 and it will save and run your script, saving you tons of keystrokes and effort over the remainder of the course.

Here is the code for each common OS. Note that it only varies in the path to the python executable.

For Windows

if (komodo.view) { komodo.view.setFocus() };
komodo.doCommand('cmd_save')
ko.run.output.kill(-1);
setTimeout(function(){
   ko.run.runEncodedCommand(window, 'C:/Python26/python -i \"%F\" {\'cwd\': u\'%D\'}');
}, 100);

For OS X

if (komodo.view) { komodo.view.setFocus() };
komodo.doCommand('cmd_save')
ko.run.output.kill(-1);
setTimeout(function(){
   ko.run.runEncodedCommand(window, 'python -i \"%F\" {\'cwd\': u\'%D\'}');
}, 100);

For Ubuntu

if (komodo.view) { komodo.view.setFocus() };
komodo.doCommand('cmd_save')
ko.run.output.kill(-1);
setTimeout(function(){
   ko.run.runEncodedCommand(window, '/opt/epd/bin/python -i \"%F\" {\'cwd\': u\'%D\'}');
}, 100);
