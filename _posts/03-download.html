In this assignment you'll download and install the software you need for this course. Don't worry if you don't know how, we'll help in the lab on Friday. But if you all try to download it during class time, our network will melt down. So download it, at least, before class on Friday.

Windows

  1. Download EPD for Windows and double-click the resulting installer file. You'll have to tell Windows it's OK to install and accept all the defaults.
  2. Log out and back in
  3. Download Komodo-Edit and double click the installer. Again, accept all the defaults.

OS X

  1. Download EPD for OS X . For me, the resulting dmg automatically mounts as a disk. If it doesn't, try double-clicking it. Then run the resulting installer.
  2. Log out and back in.
  3. Download Komodo-Edit for OS X and install it. Again, accept all the defaults.

Linux

  1. Download EPD for Linux .
  2. In a terminal window, cd to the folder where it downloaded and type: ./epd-6.3-2-rh5-x86.sh -p /opt
  3. Download Komodo-Edit for Linux , unpack the tar file with tar xzf ./Komodo-Edit-6.0.3-6811-linux-libcpp6-x86.tar.gz, cd into the resulting folder, and run the installer with ./install.sh
